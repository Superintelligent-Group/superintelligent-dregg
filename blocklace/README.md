# dregg-blocklace

DAG/blocklace consensus structures and ordering helpers.

## Owns

- Blocklace data structures.
- Causal/DAG ordering helpers.
- Consensus-facing block and checkpoint concepts.

## Does Not Own

- Node HTTP/MCP routes.
- Federation committee policy.
- Turn execution.

## Local Check

```bash
cargo check -p dregg-blocklace
```
